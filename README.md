# 🛒 Walmart Sales Analysis

## 📌 Overview

This project analyzes Walmart sales data using **PostgreSQL** for data cleaning and exploratory data analysis (EDA), and **Power BI** for creating an interactive sales dashboard.

The goal is to clean the raw dataset, analyze sales performance, and present key business insights through visualizations.

---

## 🛠️ Tools Used

- PostgreSQL
- SQL
- Power BI

---

## 🧹 Data Cleaning

The following cleaning steps were performed:

- Created a separate `walmart_clean` table.
- Removed duplicate records.
- Identified and handled missing values.
- Converted `unit_price` from text to numeric.
- Validated data types and checked for invalid values.
- Prepared the cleaned dataset for analysis.
